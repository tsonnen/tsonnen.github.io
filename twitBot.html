This is a Python scripted Twitter bot that creates new quotes, using the inspiration of Linus Torvalds and Karl Marx. More quotes will need to be added to each of the seed files, and I would like to have it create quotes that made a little bit more sense, but for now it works.

!/usr/bin/env python
# -*- coding: utf-8 -*-

import tweepy, time, sys, random, string

''' Authenication information
    To reuse this code, just change the values to be valid for your account/app'''
CONSUMER_KEY = 'abc123'
CONSUMER_SECRET = 'abc123'
ACCESS_KEY = 'abc123'
ACCESS_SECRET = 'abc123'
auth = tweepy.OAuthHandler(CONSUMER_KEY, CONSUMER_SECRET)
auth.set_access_token(ACCESS_KEY, ACCESS_SECRET)
api = tweepy.API(auth)

torvtxt = open('torvalds.txt')
torvlines = torvtxt.read().splitlines()
torvtxt.close()

marxtxt = open('marx.txt')
marxlines = marxtxt.read().splitlines()
marxtxt.close()


while 1 < 2: # Go on forever
    tweetln  = ''
    while (len(tweetln) < 28):
        torvline = random.choice(torvlines)
        marxline = random.choice(marxlines)
        torvsub = random.choice(torvline.split())
        marxsub = random.choice(marxline.split())
        if tweetln == '':
            tweetln =  torvsub + ' ' + marxsub
        else:
            tweetln  = tweetln + ' ' + torvsub + ' ' + marxsub

    print(tweetln)
    api.update_status(tweetln)
    time.sleep(900)#Tweet every 15 minutes

The source files that it draws from are just plain text with one quote per line. =
