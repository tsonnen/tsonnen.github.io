Twitter Bot

This is a Twitter bot that will use a combination of quotes from Karl Marx and Linus Torvalds to make tweets. It uses a rough Markov algorithm to combine the quotes and come up with a viable tweet.

input.txt

In order to use this, you will first need to install 'Tweepy' using pip. From there, just launch it from your terminal.
